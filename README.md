# Traffic Sign Detection
# Overview

This project presents an end-to-end Traffic Sign Detection and Classification System that combines object detection, deep learning-based classification, and web deployment. The system first detects traffic signs in images using YOLOv5 and then classifies the detected signs using a Convolutional Neural Network (CNN). To make the model accessible and user-friendly, a Flask-based web application is integrated, allowing users to upload images and receive real-time predictions.

The project demonstrates the complete machine learning workflow, including data preprocessing, annotation handling, model training, evaluation, inference, and deployment. By combining detection and classification, the system is capable of both localizing traffic signs within an image and identifying their specific categories, making it suitable for intelligent transportation systems and autonomous driving applications.

# Key Features
Traffic sign detection using YOLOv5
Traffic sign classification using CNN
Data preprocessing and annotation management
Model training and performance evaluation
Real-time image inference
Flask-based web application for deployment
Visualization of detected signs with predicted classes
End-to-end pipeline from image upload to prediction
